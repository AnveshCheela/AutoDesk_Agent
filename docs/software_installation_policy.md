# Software Installation Policy

**Document ID:** IT-POL-004  
**Effective Date:** January 1, 2026  
**Last Reviewed:** June 15, 2026  
**Owner:** IT Security & Operations — TechCorp Inc.

---

## Overview

To maintain system security, stability, and license compliance, all software installations on TechCorp-managed devices must follow this policy. Unauthorized software installations are a violation of the Acceptable Use Policy and may result in disciplinary action.

## Approved Software

TechCorp maintains a curated list of pre-approved software available for self-service installation through the **Company Portal** (Windows) or **Self Service** (macOS).

Pre-approved software includes, but is not limited to:

- Microsoft 365 suite (Word, Excel, PowerPoint, Outlook, Teams)
- Slack (Desktop & Mobile)
- Zoom (licensed version)
- Google Chrome, Mozilla Firefox
- Adobe Acrobat Reader
- 1Password (corporate vault)
- Department-specific tools (e.g., VS Code, Figma, Tableau)

The full approved software catalog is available at [https://wiki.techcorp.com/approved-software](https://wiki.techcorp.com/approved-software).

## Requesting New Software

If the software you need is not on the approved list:

1. Submit an IT ticket under **"Software Request"** at [https://helpdesk.techcorp.com](https://helpdesk.techcorp.com).
2. Include the **software name**, **vendor**, **version**, and a **business justification** explaining why the software is required.
3. The IT Security team will conduct a **security and compatibility review** (typically 3–5 business days).
4. If approved, the software will be added to the Company Portal or installed remotely by IT.

## Admin Rights Policy

- Employees are **not granted local administrator privileges** by default.
- If a task requires elevated permissions, submit a ticket for **temporary admin elevation**. Access is granted for a maximum of **4 hours** and is logged for audit purposes.
- Permanent admin rights require **Director-level approval** and are limited to engineering and IT staff.

## Prohibited Software

The following categories of software are **strictly prohibited** on all TechCorp devices:

- **Torrent clients** (e.g., BitTorrent, uTorrent, qBittorrent)
- **Unauthorized cloud storage** (e.g., personal Dropbox, Google Drive personal accounts)
- **Cryptocurrency miners** or blockchain node software
- **Hacking or penetration testing tools** (unless authorized by IT Security for approved assessments)
- **Unlicensed or pirated software** of any kind

Violations will be flagged by endpoint monitoring and escalated to IT Security and HR.

## License Management

- All software licenses are managed centrally by IT Procurement.
- Employees must **not purchase software licenses independently** or use personal license keys on corporate devices.
- License audits are conducted **quarterly**. Unused licenses are reclaimed after 60 days of inactivity.
